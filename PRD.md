# AI产业链投资研究报告 - 网页呈现 PRD

## 1. Concept & Vision

**报告定位**: 一份面向VC/PE投资人的AI产业链商业分析报告，聚焦2024-2026年AI Agent爆发背景下的产业链机会

**核心价值**: 帮助投资人快速理解AI产业链结构、识别关键投资节点、发现结构性机会

**设计调性**: 顶级咨询公司报告风格 —— 高端、克制、专业、数据驱动

---

## 2. Design Language

### 2.1 色彩系统
```
Primary:     #1A1A2E  (深墨蓝 - 主背景/标题)
Secondary:   #16213E  (深蓝 - 次级区块)
Accent:      #0F3460  (商务蓝 - 强调)
Highlight:   #E94560  (珊瑚红 - 数据/重点)
Gold:        #C9A227  (金色 - 投资机会标注)
Text:        #FFFFFF  (白色 - 主文字)
Muted:       #8B8B9A  (灰蓝 - 次要文字)
Surface:     #1E1E3F  (卡片背景)
Border:      #2A2A4A  (分割线)
```

### 2.2 字体
- **标题**: "Inter", "PingFang SC", sans-serif — 700 weight
- **正文**: "Inter", "PingFang SC", sans-serif — 400/500 weight
- **数据/数字**: "JetBrains Mono", monospace — 用于数据展示
- **中文**: "PingFang SC", "Microsoft YaHei"

### 2.3 空间系统
- 基础单位: 8px
- 大区块间距: 80px / 120px
- 卡片内边距: 32px / 48px
- 段落间距: 24px

### 2.4 动效哲学
- **克制优雅**: 微妙的fade-in (opacity 0→1, 600ms ease-out)
- **滚动驱动**: Intersection Observer 触发渐显动画
- **悬停反馈**: scale 1.02, box-shadow 增强
- **禁止**: 弹跳、闪烁、过快动画

## 4. Features & Interactions

### 4.1 导航系统
- **固定Header**: 滚动时保持可见，包含报告标题 + 章节快速跳转
- **章节锚点**: 点击跳转至对应章节，平滑滚动
- **滚动进度**: 右侧细线显示当前阅读进度

### 4.2 交互式图表
- **产业链全景图**: 悬停节点显示详情tooltip
- **投资机会矩阵**: 四象限图，悬停显示详细描述
- **数据可视化**: 关键数据使用数字动画（count-up效果）

### 4.3 内容交互
- **卡片悬停**: 轻微上浮 + 阴影加深
- **展开/折叠**: 长段落支持"阅读更多"展开
- **引用标注**: 脚注/数据来源可悬停查看详情

### 4.4 边界情况
- **加载状态**: 首屏内容优先加载，图表延迟渲染
- **空状态**: 不适用（静态内容）
- **打印优化**: 支持Cmd+P打印，保留布局和色彩

## 6. Technical Approach

### 6.1 技术栈
- **框架**: 纯HTML5 + CSS3 + Vanilla JavaScript
- **无依赖**: 不使用任何前端框架，确保轻量和快速加载
- **图表库**: 轻量自定义SVG (不引入ECharts/D3以保持简洁)

### 6.2 文件结构
```
/
├── index.html          # 主页面
├── styles/
│   └── main.css        # 所有样式
├── scripts/
│   └── main.js         # 交互逻辑
└── assets/
    └── (无外部图片，使用CSS绘制所有图形)
```

### 6.3 性能目标
- 首屏加载: <1s
- Lighthouse Score: >90
- 字体: Google Fonts预加载
- 图片: 无（纯CSS图形）

### 6.4 可访问性
- 语义化HTML标签
- ARIA标签用于交互组件
- 键盘导航支持
- 对比度符合WCAG AA标准
